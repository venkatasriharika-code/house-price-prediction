# 🏠 House Price Prediction using Linear Regression

> **Horizon Intern | Machine Learning Internship — Project B**

## 📌 Overview
A machine learning project that predicts house prices based on key features like square footage, number of bedrooms, bathrooms, house age, garage spaces, and distance to the city. Built using **Scikit-learn's Linear Regression** model.

---

## 🚀 Features
- Synthetic but realistic house dataset (500 samples)
- Exploratory Data Analysis (EDA) with scatter plots
- Feature correlation heatmap
- Linear Regression model with StandardScaler preprocessing
- Performance metrics: MAE, RMSE, R²
- Feature importance visualization
- Actual vs Predicted plot and Residual distribution
- Predicts price for a custom new house input

---
